# 使用教程

## 1. 安装依赖

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
```

## 2. 命令行运行

```powershell
.\.venv\Scripts\python.exe run_demo.py --case case_01_register_500
```

启用 DeepSeek 二次审查：

```powershell
copy .env.example .env
# 填写 .env 中的 DEEPSEEK_API_KEY
.\.venv\Scripts\python.exe run_demo.py --case case_01_register_500 --use-llm
```

## 3. 启动页面

```powershell
.\.venv\Scripts\python.exe -m streamlit run app.py
```

访问：

```text
http://localhost:8501
```

## 4. Qwen-VL 配置

如需真正分析截图内容，在 `.env` 中配置：

```text
VISION_PROVIDER=qwen
QWEN_API_KEY=你的百炼API Key
VISION_API_BASE=https://dashscope.aliyuncs.com/compatible-mode/v1
VISION_MODEL=qwen3-vl-plus
```

配置后，系统会在分析截图时调用视觉模型生成 UI 异常描述。
